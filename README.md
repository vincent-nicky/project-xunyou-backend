# 寻友（用户匹配平台） by WSJ 后端

## 一、项目介绍

能够匹配用户的移动端 H5 网站（APP 风格），基于 Spring Boot 后端 + Vue3 前端，包括用户登录、更新个人信息、按标签搜索用户、建房组队、推荐相似用户等功能。

## 二、技术选型

Java SpringBoot 框架

MySQL 数据库

MyBatis-Plus

MyBatis X 自动生成代码

Redis 缓存

Redis 分布式登录

Redisson 分布式锁

Easy Excel 数据导入

Spring Scheduler 定时任务

Swagger + Knife4j 接口文档

Jaccard相似度匹配用户
